c언어는 함수 지향 프로그래밍
c++는 객체 지향 프로그래밍


class = 구조체 = 데이터 타입

struct 대신 class 사용

변수와 함수도 사용가능

변수는 접근을 차단해야함   (캡슐화/데이터 은닉성)

함수는 상관 없다.

class 멤버변수는 외부에서는 사용 불가

class의 public 에 있는 멤버함수로 멤버변수 사용 가능
private에 있는 변수는 public에 있는 함수를 통해 호출



스택은 LIFO알고리즘

class는 키워드를 쓰지 않으면 private로 자동처리

같은 class로 만든 구조체는 대입 가능
